LemmBox - Features

What is inside the starbox?

aka all the stuff this mod has

  • Tempo slider goes up to 1000BPM
  • Expanded octave range from -8 to +8 octaves
  • High-quality chipwaves:
    • HD Rounded
    • HD Triangle
    • HD Sawtooth
  • A custom chipwave fromLemmBox "Trapasquare"
  • New theme "StarBox Dark"

    (Just called "StarBox" in the theme selector as there is no StarBox light mode yet
